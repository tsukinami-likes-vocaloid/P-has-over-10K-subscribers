このサイトについて
このサイトについて、Q & A 形式で説明します。
このサイトはどんなサイト？
YouTubeにて、チャンネル登録者数が1万人を超えているボカロPさんを検索できるサイトです
作ろうと思ったきっかけは？
暇だったので、登録者が1万人超えてる方を全員集計してたのですが、「どうせなら検索できるツールもつくりたいよな...」となり、このサイトが誕生しました
コードは自分で書いたの？
ほとんどのコードはChatGPTさんか、優しい方がインターネットにあげてくれている、フリーテンプレのコピペです。そのため、詳しい人が見たら頭痛がするような構成＆コードになっています。この分野に詳しい友達に見せたら「ハゲそう」って言われました
テンプレからもってきたって言ってたけど、具体的にお世話になったサイトは？
CSS Stock 様、 CSS box-shadow examples 様には本当にお世話になりました...感謝申し上げます...
これ1人でつくってるの？
1人（ちょっと友達に手伝ってもらってる）です。私はhtmlなりcssなりjavaなりの知識が乏しいので、これらに詳しい＆手伝ってやってもいいぜって方がいたらTwitterのDMに来てくれるとうれしいです...。また、ボカロPさんのタグ付け作業などもあるので、「Webページ制作はよくわからんけど、ボカロは好き！」という方も、一旦Twitterを見てくださるとうれしいです...！
検索の仕組みは？
それぞれのボカロPさんに、「アーティスト名」「アーティスト名のよみがな」「タグ」のデータが保存されています。検索の際は、「アーティスト名を入力する」、「ひらがなでよみがなを入力する」、「タグで絞り込む」の3つを駆使してみてください。
集計方法と対象は？
対象は、「登録者が1万人を超えている日本人広義ボカロPさん」、です。具体的には、登録者が今日（2024/06/18くらい）時点で1万人を超えている&投稿しているボカロ曲が2曲以上、の方を集計しています。（ 林田匠さんなど、すべての投稿楽曲が2曲で、そのうち1曲がボカロ...などの例外は除きます。）広義ボカロを対象としているので、マサラダさんなどVOCALOIDを使用していない方も集計しています。方法は、完全に手動です。スクレイピングとかも考えましたが、知識がない私がやってサーバーに負荷とかをかけたくなかったので...
タグなんかおかしいけど...？
まだテスト段階のため、ダミーのタグを表示させています。バグも多分ありますし、UIもあんまりよろしくないです。申し訳ない...
